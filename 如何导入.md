# 🚀 如何导入案例数据（最简单的方法）

## 方法 1：使用 PowerShell 脚本（推荐）

### 步骤：

1. **双击运行**：`一键导入.ps1`

2. **输入 Supabase Service Role Key**：
   - 打开 https://supabase.com/dashboard
   - 选择项目 **showcase-website**
   - 进入 **Settings** → **API**
   - 复制 **service_role** key
   - 粘贴到提示框中

3. **等待导入完成**

---

## 方法 3：直接双击批处理文件

双击 `run-import.bat`，然后输入 Service Role Key。

---

## 💡 重要提示

### 获取 Supabase Service Role Key：

1. 访问 https://supabase.com/dashboard
2. 选择项目 **showcase-website**
3. 点击左侧菜单 **Settings** → **API**
4. 在 **Project API keys** 部分，找到 **service_role** key
5. 复制这个 key

⚠️ **注意**：Service Role key 具有完全访问权限，请妥善保管，不要泄露。

## ❓ 遇到问题？

如果导入失败，请检查：

1. ✅ Service Role Key 是否正确
2. ✅ Supabase 项目是否正常运行
3. ✅ 数据库表是否已创建（`cases` 表）
4. ✅ 网络连接是否正常
